  • Using Bbootstrap 4 with Django CMS:

    - Follow the installation instructions in the following link: Github: djangocms-bootstrap4. Do not run python manage.py migrate. NOTICE: there is an incompatibility issue.
    In the project directory (academycity), Run the command: pip install -r requirements.txt. Open the file requirements.txt and change the line with psycopg2 to psycopg2==2.7.7 (see: issues/6666)
    Run the commend: python manage.py migrate
    Run the server with the command: python manage.py runserver
