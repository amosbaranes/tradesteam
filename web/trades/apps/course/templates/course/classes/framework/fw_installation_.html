    Create a virtual machine and upload the project

    1 - Open docker and create virtual machine called trades:
        docker-machine create -d virtualbox trades
    2 - Make the virtual machine active
        eval $(docker-machine env trades)
    3 - View the virtual machines on your computer and notice the * next to the trades machine
        docker-machine ls
    4 - On docker you should be in the web folder. issue the command:
        cd /c/envs/projects/trades
    5 - On docker issue the following command to create the services:
        docker-compose build
    6 - Upload the services in the virtual machine (VM):
        docker-compose up -d
    7 - Run migrate on the MV to create the database:
        docker-compose run --rm web //usr/local/bin/python manage.py migrate
    8 - Run collect static files
        docker-compose run --rm web //usr/local/bin/python manage.py collectstatic --no-input
    9 - Create super user:
        winpty docker-compose run -- rm web //usr/local/bin/python manage.py createsuperuser
        When prompted, enter the following:
        User: admin
        email: admin@gmail.com
        Password: sql1pass
    10 - Find the IP of your VM:
        docker-machine ls
    11 - open the browser and type the IP address of your VM with port 8080
    12 - Register yourself as student:
